# 道术将为天下裂：清中叶“汉宋之争”的一个思想史研究

张循著，2017年

### 1. 研究视角的创新：社会史与思想史的结合
作者不满足于以往仅从“政治高压”（如朱维铮的观点）或“新义理的兴起”（如张寿安的观点）来解释汉宋之争。他提出了“眼光向下”和“眼光向上”两个维度的结合：
*   眼光向下（社会/风气层面）： 关注普通读书人（科举士子）。
*   眼光向上（思想/学术层面）： 关注专业的精英学人。

### 2. 汉宋之争的社会层面：科举与风气
作者认为，“汉宋之争”不仅仅是学术争论，更是一种弥漫一世的社会“风气”。
*   不读书也争汉宋： 大量普通读书人并不真正懂汉学或宋学，但依然卷入争论。
*   科举的因素： 许多士人因为厌恶科举考试中僵化的“宋学”，从而在情感上倾向“汉学”；或者为了在科举中炫耀博学，在策论中堆砌汉学考据。这种对科举的态度导致了盲目的“反宋”风气。

### 3. 汉宋之争的思想层面：汉学的内在紧张（核心观点）
这是全书最核心的论点。作者聚焦于专业学者，并认为，思想史意义上的“汉宋之争”并非简单的两派对立，而是儒学发展到清代汉学阶段后，自身内部产生的破裂。
*   第一重含义（外在）： 汉学（重考据、重知识）与理学（重德性、重修养）的对立。
    * 汉学：注重“由字通经，由经明道”，强调通过知识积累（小学、训诂）来接近圣人之道。特征是“学人”。
    * 宋学（理学）：被视为“尊德性”的传统。注重身心修养、道德实践，认为“理”具于心，不完全依赖书本知识。特征是“儒者”。
*   第二重含义（内在）： 所谓的“汉宋之争”，在清代后期其实大都是“汉学内部”的紧张关系。
    * 传统儒学要求“尊德性”与“道问学”统一，要求“考据”与“义理”统一。但清代汉学过度发展了“考据”兴趣，导致考据与义理、知识与德行发生分离。
*   真正的矛盾： 汉学家既难以抑制对考据的痴迷，又在理论上无法放弃儒家“明道/致用”的宏大目标，这种理论与实践的脱节，构成了汉宋之争的深层焦虑。

### 4. 宋学家的“汉学化”
作者通过分析方东树等“宋学”卫道士发现，清代的“宋学家”在批评汉学时，往往也不自觉地接受了汉学的学术范式（如重视考据、训诂）。
*   以汉攻汉： 所谓的汉宋之争，在很大程度上其实是“汉学内部”的争论。连宋学的支持者也变成了考据学家，他们共享着同样的学术话语体系。

### 5. 汉宋调和与现代学术的兴起
书中讨论了陈澧等人试图调和汉宋的努力，但认为是失败的，因为考据与义理的分裂已不可挽回。

* 陈澧虽然主张“汉宋兼采”，但他所理解和接受的“宋学”，并不是宋明理学中核心的心性、天理等形而上学内容。
    *   把朱子“汉学化”： 陈澧推崇朱熹，是因为他认为朱熹注重经典注疏，符合汉儒家法（书中提到他列举大旨为“明朱子之为汉学”）。
    *   回避理学核心： 他对周敦颐、二程的形而上学不感兴趣，甚至认为无用。
    *   陈澧所谓的调和，实际上是在汉学内部强调“考据”与“义理”（经义）的结合，而不是真正沟通了“清代考据学”与“宋明理学（德性之学）”。他本质上仍是一个汉学家，只是想补救汉学琐碎的弊病。
* 考据已经事实上独立，大势已不可逆转
    *   理论上的完满与现实的断裂： 在儒家传统理论上，“由考据（小学）通向义理（大道）”是天经地义的。但在当时的学术实践中，考据学已经高度专业化、技术化（如音韵、训诂），学者们往往终身埋首于此，不再（也无力）上升到“义理”层面。
    *   陈澧的无力感： 陈澧试图用理论强行将二者拉回一体，但在实践中极为困难。他自己也感叹“只开得基址颇大（考据），而不能起屋（义理）”。
* 两边不讨好，缺乏学术认同。陈澧的调和论在当时并没有得到真正的认可，处境尴尬。
    *   汉学家的批评： 纯粹的汉学家（如后来的章太炎、刘师培）认为他牵强附会，搞“鸠合汉宋”，是“揣豪于千马”（在千匹马里找一根长毛硬说是一样的），对他评价不高。
    *   宋学家的隔膜： 真正的理学家（如倭仁等）认为他还是在讲考据，并未真正切入身心性命的修养。
    *   书中描述陈澧晚年感到“所学不可以语人”，有一种深刻的孤独感。这说明他的调和方案在当时已经失去了现实的土壤。
*   最终结局： 这种“调和的失败”最终导向了清末民初的学术分科。原本纠缠不清的“考据”变成了现代的语言学、历史学；“义理”变成了现代的哲学。
*   意义： 西方学科体系进入中国后，迅速接管了这一局面。陈澧试图在一个旧框架内弥合裂痕，而历史的解决方式是彻底拆解这个框架。
*   结论： “汉宋之争”实质上是中国学术从传统向现代转型的前奏，是儒学解体、专门之学（为知识而知识）兴起的阵痛。

### 附录：蒙文通个案研究


书末附录通过分析蒙文通的经学历程，展示了一位处于新旧时代交替中的学者，如何在汉学（考据）与宋学（义理）之间挣扎与调适，进一步佐证了正文关于学术转型期“内在紧张”的观点。
